# Proyecto-Dise-o-de-Pruebas-Urban-Routes-
Análisis de requisitos para función de compartir automóvil y licencia de conducir en Urban Routes

•	Objetivo: Análisis de requisitos para la función de compartir automóvil y licencia de conducir.
> 
•	Herramientas: draw.io, Google Sheets, técnicas de diseño de pruebas, análisis de requisitos y documentación técnica.
>
•	Casos de prueba: 12 casos de prueba usando clases de equivalencia y valores límite; identificación de 3 zonas grises. Elaboración de mapa mental, diagrama de flujo
> 
•	Resultados: 100% de cobertura de lógica de cálculo.
